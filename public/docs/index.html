Introduction

Base URL: http://calabozos.test 
This documentation aims to provide all the information you need to work with our API.

<aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

POST api/login

Example request:
curl --request POST \
    "http://calabozos.test/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"gbailey@example.net\",
    \"password\": \"|]|{+-\"
}"

const url = new URL(
    "http://calabozos.test/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "gbailey@example.net",
    "password": "|]|{+-"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Retrieve and synchronize all character classes from the API.

This endpoint fetches all classes from the external API and ensures they are stored in the local database through the ClassService.

Example request:
curl --request GET \
    --get "http://calabozos.test/api/calabozos/classes" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://calabozos.test/api/calabozos/classes"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
